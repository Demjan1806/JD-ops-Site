# Project Instructions

Build a professional one-page website for JD Ops.

Inspiration: https://www.infrastructureinnovators.nl/

Style:
- Dark professional look
- Clean hero section
- Simple navigation
- Stats section
- Services section
- About section
- Contact section
- Footer
- Mobile responsive

Do not copy the example website exactly.

Tech:
- Next.js App Router
- TypeScript
- Tailwind CSS

Rules:
- Keep it simple
- Use reusable components
- Keep text easy to edit
- No database yet
- No login yet
- Beginner-friendly code
