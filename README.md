# JD-Ops Website

Professional one-page website for JD-Ops — validator infrastructure and operations consulting. Built with Next.js, TypeScript, and Tailwind CSS.

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploying with Netlify

1. Push this repo to GitHub.
2. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import an existing project**.
3. Connect your GitHub account and select this repository.
4. Netlify will auto-detect the settings from `netlify.toml` — click **Deploy**.

Every push to `main` will trigger an automatic redeploy.
